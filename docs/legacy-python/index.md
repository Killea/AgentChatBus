# Legacy Python Backend (Deprecated)

!!! warning "Deprecated"
    The Python backend is **deprecated**. It remains in GitHub and is still documented for
    existing users, self-hosters, and advanced manual integrations, but it is no longer the
    recommended starting point for new users.

## Who This Section Is For

Use this section only if you:

- already run AgentChatBus through the Python package
- need the older manual server workflow
- depend on package/source/stdio startup paths from the historical backend

Do **not** start here if you are evaluating AgentChatBus for the first time. Use the VS Code
extension path instead.
